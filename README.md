FoodHub — Modern Food Ordering Website

A professional, responsive food delivery/restaurant website built with Bootstrap 5, HTML, CSS, and JavaScript.

Features:
- Homepage with hero section, featured foods, and categories
- Full menu page with food items, ratings, and category filtering
- Special offers page with promotional cards
- Shopping cart with order summary
- Responsive design (mobile, tablet, desktop)
- Modern UI inspired by top food delivery platforms like Zomato & Swiggy
- Bootstrap 5 CDN (no build step required)
- Font Awesome icons throughout

Pages:
- index.html — Homepage with featured foods and quick overview
- menu.html — Full menu with category filtering
- offers.html — Special promotions and discounts
- cart.html — Shopping cart and order summary
- about.html — About FoodHub and stats
- contact.html — Contact form and business info

Files:
- css/style.css — Custom food theme styles (complement Bootstrap)
- js/main.js — Menu filtering and form handling
- Images use placeholder URLs (replace with real food images)

Quick Start:
1. Open index.html in your browser
2. Click "Order Now" or browse Menu
3. Add items to cart
4. Proceed to checkout

Deploy:
- GitHub Pages: Push to repo and enable Pages
- Netlify/Vercel: Drag-and-drop or connect repo
- Any static hosting: Upload the folder

Customization:
- Replace placeholder images with real food photos
- Update restaurant name, pricing, and menu items
- Add real payment gateway integration
- Connect to a backend for orders management
- Customize colors in css/style.css (--danger color, etc.)
